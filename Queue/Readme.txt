------------Queue-------------
It is a linear data structure in which data is stored from rear end and is deleted from front.
    Example : A line for movie ticket First person will get ticket first 

It can be implemented by :
    1. Array 
    2. Linked list

Different Operations are :
    1. Enqueue data
    2. Dequeue data
    3. Peek 
    4. IsEmpty 

Types of Queue are :
    1. Dequeue(Double ended queue)
    2. Priority Queue  

T.C : 
    Enqueue-> O(1)
    Dequeue-> O(1)
    IsEmpty-> O(1)
    
S.C : O(N)
