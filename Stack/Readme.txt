-----------------Stack---------------------

It is the linear data structure in which insertion and deletion of elements take place only from one end.
    Example : In a Piles of Plate, last plate will be picked first.

It can be Implemented By :
    1. Using Array
    2. Using LinkedList

Operations : T.C
    1. Push  0(1)
    2. Pop   0(1)
    3. IsEmpty 0(1)
