----------------LinkedList-----------------
Linkedlist 
It is a linear data structure that store data in a non contiguous format.
It consists of node that consists of data and adress part.

Features of Linkedlist
    1. Dynamic Memory Allocation
        Memory is allocated in Runtime
    2. Insertion and deletion is very easy since no element are to be       inserted  shifted and only links are to be updated.
    3. efficient memory management

Types Of Linkedlist
    1. Singly linked-list
    2. doubly linked-list
    3. circular linked-list 

Disadvanges of Linkedlist
    1. No random acess
    2. wastage of Memory

T.C - O(n)
S.C - O(n)
